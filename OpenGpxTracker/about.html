Open GPX Tracker v1.1

Created by Juan M. Merlos @merlos

This app was created to help Open Street Map contributors to generate Public GPX traces.

Open GPX Tracker is open source and the code is distributed under GPL license. It is available on GitHub

Tip! Long press over the map to add a Waypoint.
